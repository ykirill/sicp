## Упражнение 1.11

Функция f определяется правилом:
f(n) = n, если n < 3, и
f(n) = f(n -1) + f(n - 2) + f(n - 3), если n >= 3.
Напишите процедуру, вычисляющую f с помощью рекурсивного процесса.
Напишите процедуру, вычисляющую f с помощью итеративного процесса.
